## Introduction

The setup includes a variety of sensors to observe the plant and the environment. It then uses this data to maintain and grow plant to its fullest potential.

## Hardware Required

- Raspberry Pi x 2
- Humidity Sensor
- PiCam
- LDR sensor
- Temperature sensor
- Solenoid valve
- Full spectrum lights
